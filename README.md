# Belief-Changer

An open-source, nonprofit effort to generate high-quality, **Allen Carr "Easyway"-style belief-change books** for the many behaviors no such book exists for yet — gaming, doom-scrolling, pornography, and more.

## The core idea

People already choose what they believe is their happiest option in the moment. The problem is that those beliefs are often misaligned with reality — we price in the immediate payoff and silently zero out the real costs. **Correct the belief model and the behavior changes on its own**, without willpower or a sense of sacrifice. That is the mechanism Easyway and *The Freedom Model* exploit, and the one this project aims to generalize and scale with AI.

## Where things stand

- **`docs/VISION.md`** — the living vision & working tracker (philosophy, generation pipeline, orchestration, open questions, north star). Start here.
- **`prompts/`** — prompts for the agent pipeline (e.g., the book-analysis sub-agent).

This repository is early. The current focus is the MVP: a research → master-plan → chapter-by-chapter generation pipeline, prototyped inside an agent orchestrator.

## Principles

- Capture the **method** (how belief change actually happens), not just the prose style.
- **Non-shaming** framing throughout — escape a trap, never a lecture.
- Write **original** prose; learn the mechanism, don't reproduce copyrighted text.
- Optimize for **quality** over cost on the foundational, one-time work.
