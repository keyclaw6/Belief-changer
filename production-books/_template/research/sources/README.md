# Sources (raw dumps)

Dump **any** raw source material here — **one file per source, any format** (`.md`, `.txt`, `.html`, `.json`, pasted forum threads, scraped pages, video transcripts, study text). Don't clean or synthesize on the way in — the point is to gather a lot, fast.

**Agent-gathered findings also live here.** The `belief-changer-deep-research` skill writes source-level structured files (e.g. `exa-<topic>.md`) into this folder — still source-level, not synthesized — so they sit alongside the human dumps. The synthesis step digests everything here.

Later, a synthesis step digests everything in this folder into the two curated files one level up:
- `../lived-experience.md` — experiences, reasons, shame, testimonials
- `../scientific-evidence.md` — studies & mechanisms (supported vs contested)

So this folder is the **raw input**; those two are the **synthesized output**.

## Naming
Descriptive, lowercase-hyphenated, type-tagged — e.g. `reddit-nofap-thread-2024-03.md`, `study-kuhn-gallinat-2014.txt`, `ybop-faq-scrape.html`, `yt-testimonial-transcript-abc.txt`.
