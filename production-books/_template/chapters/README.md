# Chapters

Finished chapters land here as `chapter-01.md`, `chapter-02.md`, … — built up **one (or two) at a time**, in order.

Each chapter-writer receives exactly three things (the anti-repetition context strategy):
1. The **master plan** (`../master-plan.md`) — whole-book oversight.
2. The **immediately previous chapter only** — local continuity.
3. The **style guide** (`/prompts/style-guide.md`) — voice & method.

A reviewer pass critiques each chapter against the master plan + style guide; the writer revises until accepted; then the next chapter begins.
