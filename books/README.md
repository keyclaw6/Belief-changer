# Source books (analysis inputs)

Upload the source **PDFs** to analyze here. Current set — 3 books, all PDF, ≤ 100 pages each:

- `easyway.pdf` — Allen Carr's *Easyway*
- `freedom-model.pdf` — *The Freedom Model*
- `<third-book>.pdf` — title TBD

**Naming:** lowercase, hyphenated, `.pdf` (e.g. `easyway.pdf`).

**How they're used:** the book-analysis pipeline reads each PDF chapter by chapter and writes **one analysis document per book** to `../analysis/<book-slug>.md`.

**Two ways to get a book here:**
1. Drop the PDF in this folder (commit it), or
2. Attach the PDFs in the chat and they'll be placed here for the run.

> Note: this repo is intended to become public. If you'd rather not commit the raw PDFs, use option 2 — attach them in chat and they'll stay out of git.
